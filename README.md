# ALGo (Front-End)

- "ALGo"는 식품에 알레르기가 있는 사람들뿐만 아니라 식품의 알레르기 정보를 얻고자 하는 사람들에게 식품 정보를 제공해주는 웹 사이트 입니다.
- 본 프로젝트의 프론트엔드(Front-end)는 Vue.js 3.0으로 개발되었습니다.

## 프로젝트 빌드 (Development)
```
npm install
npm run dev -- --host
```

## 프로젝트 빌드 (배포버전)
```
npm install
npm run build
npm install -g serve
serve -s dist
```

## 기술스택
- Vue.js 3.0
- HTML
- JavaScript
- CSS
